## Intro
This program contains the python CNN trained model for skin cancer detection with its GUI.CNN model is trained on bunch of images containing skin cancer and normal wart is capable of differentiating between them. GUI part enables users to just simply type the name of image in the image folder and it can easily detect if the image is of skin cancer or wart using backend algorithms making it easy for normal people to use.Makeathon Thapar 2020.
